# Mips32
基于仿真的 32 位虚拟计算机设计与实现
Mips32指令系统
实现内容：
  - 完成Mips32指令的取指、译码、计算、访存和写回五个步骤的软件模拟
  - 能够像系统输入机器语言源程序
  - 能够对内部寄存器进行初始化
  - 能够运行程序
  - 能够查看运行结果，能够反映指令的执行过程
  - 模拟五段流水线的执行过程
  - 解决了数据相关问题
  - 反映了流水线的执行过程
  - 完成了图形界面的开发
